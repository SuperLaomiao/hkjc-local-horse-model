### 助手（最终回复）

可以，我刚查了香港赛马会的官方资料和一些经典研究。先给你结论：

**没有“必胜模式”。**
香港赛马会赛马主要是彩池制，不是庄家固定赔率。马会官方写明：独赢、位置、连赢、位置Q、孖宝等派彩池只派出 **82.5%**，三重彩、四连环、四重彩、三宝等常见组合池派出 **75%** 左右。也就是说，普通人一下注，长期期望值先天然少了约 **17.5% 到 25%**。你要赚钱，不是“猜中”就够，而是要比整个市场更准，准到足以覆盖抽水、赔率变动和下注冲击。来源：[HKJC Horse Racing Betting Guide](https://is.hkjc.com/AOSBS/help/en/HR_Guide.html)

**现在香港赛马市场仍然很大，也很专业。**
2024/25 赛季有 **88 个赛马日、847 场香港本地赛事、393 场海外转播赛事**，赛马投注额 **HK$1388.5 亿**，比上一季升 3%。来源：[HKJC 2025/26 Racing Media Guide](https://campaign.hkjc.com/media-guide/pdf/media-guide-eng-2025-26.pdf)。截至 2026-06-07 的官方统计，热门马共 763 匹次，赢出 219 次，胜率 **29%**，跑入前三 459 次，位置率 **60%**。来源：[HKJC Jockeys’ Favourite Statistics](https://racing.hkjc.com/en-us/local/information/jockeyfavourite)

别人真正专业的做法，大致是这几类：

1. **量化模型派**：像 William Benter 早期香港赛马系统，用历史赛果、马匹评分、负磅、档位、骑师、练马师、路程、场地、近期状态等估算每匹马真实胜率，再和市场赔率比较。只在“模型概率 × 派彩 > 1”时下注。来源：[Benter, Computer Based Horse Race Handicapping and Wagering Systems](https://gwern.net/doc/statistics/decision/1994-benter.pdf)

2. **赔率价值派**：不问“哪匹最可能赢”，而问“哪匹被市场低估”。例如一匹马真实胜率你估 20%，但市场赔率像是只给它 12% 的机会，这才可能有价值。

3. **资金管理派**：用 Kelly 或分数 Kelly 控制注码。Benter 也提醒全 Kelly 很危险，模型只要高估优势，就可能大幅回撤。

4. **专攻彩池派**：一些人不碰独赢，改研究连赢、位置Q、三重彩、四重彩，因为组合池有时更容易出现公众误价。但这些池抽水更高，波动也更大。

5. **临场赔率/尾注派**：看临场资金流向，因为最后几分钟大额资金会改变赔率。问题是香港市场很成熟，很多专业资金也在看同样的东西。

所以我的判断是：
**“必胜模式”没有；“可测试的正期望模型”理论上有，但门槛非常高。** 对普通人来说，最现实的研究方向不是买别人所谓心水，而是建立一套回测框架：拿多年 HKJC 赛果和派彩，测试每个规则扣除抽水后是否仍然赚钱。任何没经过回测、只讲命中率的策略，基本都不可信。命中率高不等于盈利，赔率不够照样输钱。

### 助手（最终回复）

可以，而且这是最正确的切入点之一：**先做“马匹能力/状态模型”，再谈下注。**

但要修正一个小点：马本身不是完全独立的。它的能力有独立部分，比如速度、耐力、健康、状态、适应距离；但比赛结果是相对的，还受对手、步速、档位、负磅、骑师、场地、当天跑道偏差影响。所以模型应该是：

**先估每匹马今天能跑出什么水平，再把全场马匹放在一起算胜率。**

一个专门研究马匹的模型可以这样做：

1. **马匹基础能力**
   用历史赛绩、完成时间、负磅、班次、路程、场地、头马距离等做一个“能力评分”。HKJC 马匹档案里有这些字段，例如赛绩、路程、班次、档位、评分、骑师、练马师、实际负磅、沿途走位、完成时间等。
   来源：[HKJC Horse Profile 示例](https://racing.hkjc.com/racing/information/English/Horse/Horse.aspx?HorseId=HK_2025_L913)

2. **分段时间模型**
   不只看最终名次，要看每匹马怎么跑：前段快不快、中段能不能维持、末段有没有冲刺。HKJC 的 form guide 说明有马匹分段时间，通常分为起步至 800m、800m 至 400m、400m 至终点。
   来源：[HKJC Form Study Help](https://www.hkjc.com/english/formguide/formstudy_help.asp)

3. **状态模型**
   看近 3-5 场走势、休赛天数、体重变化、试闸表现、兽医记录、是否换配备。香港数据优势就在这里：马匹体重、试闸、兽医记录、沿途走位都相对公开。

4. **路程/场地适应**
   有些马 1200 米强，1400 米就弱；有些喜欢田草，有些适合泥地；有些擅长好地，不适合软地。这个要给每匹马建“适应曲线”。

5. **跑法/步速匹配**
   马有领放、跟前、居中、后上之分。如果一场里太多快马，前段可能互相烧死，后上马反而有利。这个不是单马因素，但必须接到马匹模型后面。

6. **概率输出**
   最后不要输出“买 A 必赢”，而是输出：
   - A 马胜率 18%
   - 跑入前三 43%
   - 公允独赢赔率约 5.6 倍
   - 当前市场如果给到 8 倍，才可能有价值

真正有用的不是“预测冠军”，而是找出：**市场低估了哪匹马。**

所以答案是：**可以做，而且应该做成一个马匹数据库 + 能力评分 + 状态评分 + 胜率模型。**
下一步我建议先做一个最小版本：只抓 HKJC 近几年赛果和马匹档案，先建“马匹速度评分”和“近况评分”，不急着下注，先回测看看扣除抽水后是否还有正收益。

### 助手（最终回复）

可以，**开放资料足够建立一个“可回测的马匹模型”**，但还不够直接证明能稳定盈利。

香港赛马会公开资料其实很适合做模型，尤其比很多地区更规范。能用的数据大概分几层：

**可以建模的数据**
1. **赛果数据**：名次、完成时间、头马距离、班次、路程、场地、档位、负磅、骑师、练马师、独赢赔率、沿途走位。
   来源：[HKJC Race Results](https://racing.hkjc.com/en-us/local/information/localresults)

2. **马匹历史档案**：每匹马过往赛事、评分、血统、赛绩、总奖金、练马师/骑师记录。
   示例：[HKJC Horse Profile](https://racing.hkjc.com/racing/information/English/Horse/Horse.aspx?HorseId=HK_2025_L913)

3. **分段时间**：可以分析马是前速型、均速型、末段冲刺型，避免只看最终名次。HKJC form guide 说明有马匹分段时间。
   来源：[HKJC Form Study Help](https://www.hkjc.com/english/formguide/formstudy_help.asp)

4. **试闸/晨操/兽医/健康记录**：这些对“今天状态”很关键。HKJC Racing Info 页面有试闸、晨操、兽医记录、最新马匹消息等入口。
   来源：[HKJC Racing Information](https://racing.hkjc.com/en-us/local/information/selecthorse)

5. **派彩和赔率**：用来判断模型有没有“价值投注”。没有赔率，模型只能预测；有赔率，才能回测是否赚钱。
   来源：[HKJC Betting Guide](https://is.hkjc.com/AOSBS/help/en/HR_Guide.html)

**可以做出的模型形态**
最现实的第一版不是 AI 神经网络，而是一个结构化评分模型：

```text
马匹今日评分 =
基础速度评分
+ 路程适应
+ 场地适应
+ 班次强度调整
+ 负磅调整
+ 档位影响
+ 近期状态
+ 分段走势
+ 试闸/健康信号
+ 骑师练马师组合
```

然后把一场比赛所有马的评分转成概率：

```text
A 马胜率 18%
B 马胜率 15%
C 马胜率 12%
...
```

最后再跟市场赔率比较：

```text
模型认为 A 马合理赔率 = 1 / 18% = 5.56 倍
市场独赢如果是 8 倍，可能有价值
市场独赢如果是 4 倍，就不买
```

**关键限制**
最大的难点不是资料不够，而是三个：

1. **数据清洗难**：HKJC 页面公开，但不是全部都有方便 API，要抓取、标准化、去重、处理改名/退役/海外赛绩。

2. **赔率会反映公众智慧**：香港赛马市场很成熟，热门马、骑师、档位、近况通常已经反映进赔率。

3. **抽水很高**：独赢/位置等常见池长期要先克服约 17.5% 抽水，部分组合池更高。模型必须明显比市场准，才有正期望。

所以结论是：
**能建立，而且资料基础是够的；但应该先做“研究模型 + 回测系统”，不要一开始就当成赚钱机器。** 第一阶段目标应该是验证：模型在历史数据里，扣除抽水后，是否能找到稳定的正期望下注点。

香港本地赛马模型可以做成三层：

**第一层：马匹能力模型**

核心是判断一匹马“本身有多强”。

可用资料包括：

- 历史名次
- 完成时间
- 头马距离
- 班次
- 评分
- 负磅
- 路程
- 场地
- 档位
- 沿途走位
- 马匹体重
- 独赢赔率
- 骑师、练马师
- 配备变化

这些 HKJC 本地赛果和马匹档案里基本都有。

**第二层：今日状态模型**

同一匹马不是每天一样强，要看今天状态：

- 近 3-5 场走势
- 上次出赛距今多少天
- 最近是否加分/减分
- 体重是否异常变化
- 是否换骑师
- 是否换配备
- 是否试闸表现好
- 是否有兽医记录
- 是否升班/降班
- 是否转场地、转路程

这一层很重要，因为赛马不是静态数据，马的状态波动很大。

**第三层：比赛环境模型**

马强不强是一回事，今天这场合不合适是另一回事：

- 沙田 vs 跑马地
- 草地 vs 全天候泥地
- 1000m、1200m、1400m、1600m 等路程适性
- 内档/外档影响
- 当日跑道偏差
- 快马多不多，步速会不会太快
- 这匹马是领放、跟前、居中还是后上
- 班次强弱
- 对手整体水平

最后模型输出的应该是概率，而不是一句“买几号”：

```text
A 马胜率：21%
B 马胜率：17%
C 马胜率：13%
D 马胜率：10%
...
```

然后再和赔率比较：

```text
A 马模型合理赔率 = 1 / 21% = 4.76 倍

如果市场独赢赔率是 3.5 倍：不买
如果市场独赢赔率是 7.0 倍：可能有价值
```

所以结论很明确：

**能搞，而且只搞香港本地赛马是一个合理范围。**

但要分清两件事：

1. **预测模型**：判断哪匹马更可能赢。这个可以做。
2. **盈利模型**：扣除马会抽水后仍然长期赚钱。这个难很多，必须回测验证。

最现实的第一版可以这样做：

```text
HKJC 本地赛果数据
-> 建马匹历史数据库
-> 做速度评分/状态评分/场地路程适性
-> 每场赛前输出胜率
-> 和独赢/位置赔率比较
-> 回测 ROI
```

我会建议先做一个 **香港本地马匹评分模型 MVP**，不要一开始搞太复杂。先验证一个问题：

**只靠公开资料，我们能不能比市场赔率更早、更稳定地发现被低估的马。**

```text
数据模型 → 赛前概率 → 赔率价值判断 → 小注策略 → 赛后复盘 → 持续校准
```

你到时候投它推荐的东西，本质上是执行一个 **概率投注策略**，不是买基金、不是理财、不是保证收益方案。

怎么用才对：

1. **只投模型标记为 Value 的马**
   不是模型第一名就买，而是要同时满足：
   - 模型胜率够高
   - 市场赔率比模型公允赔率更划算
   - Value Edge 为正
   - 注码不超过 bankroll 上限

2. **每场不一定下注**
   最好的策略不是每场都买。很多场应该显示 `PASS / 观望`。真正像样的系统，是敢于不下注。

3. **用固定资金池**
   比如你设一个测试池：
   ```text
   总 bankroll：HK$1,000 或 HK$10,000
   单场最多：1% 到 1.5%
   不加码追输
   不临场情绪下注
   ```
   这才叫策略。

4. **每周看结果**
   看三件事：
   - 推荐马命中率
   - 实际 ROI
   - 模型预测胜率和真实胜率是否接近

5. **再校正模型**
   如果模型说某类马胜率 20%，但长期只赢 8%，说明模型高估了。
   如果模型说 12%，但长期赢 18%，说明它低估了。
   我们就根据这个调参数。

```text
R5 沙田 1200m

市场赔率：6.5

```

```text
投注：HK$100

盈利：HK$550
模型累计 ROI：更新
```

**我的建议：开跑前 10 到 5 分钟下，最合理。**

原因是香港本地 WIN / PLACE 这类主要是彩池制，不是固定赔率。香港马会说明本地彩池是大家投注后按中奖组合分派派彩，所以赔率会一路变，越接近开跑，赔率越接近真实市场。固定赔率只有部分 bet type 才是下注时锁价。

实操节奏我建议这样：

1. **T-24h 到 T-3h：只做模型预测，不下注**
   等官方 race card、排位、负磅、骑师、场地资料稳定。

2. **T-60 到 T-30 分钟：筛选候选马**
   看模型胜率、排位、场地、赔率是否有 value，但还不急着下。

3. **T-15 分钟：第一次确认赔率**
   如果模型给 20% 胜率，公平赔率约 5.0；市场赔率如果明显高于 5.0，才有 value。

4. **T-10 到 T-5 分钟：最佳下注窗口**
   这个时候资金池比较成熟，赔率没早段那么虚；同时又不会拖到最后几十秒系统卡、赔率突变、来不及取消判断。

5. **T-3 分钟以内：只适合小注或放弃**
   最后大户资金可能突然进来，赔率会被砸低。你以为买到 8 倍，最后可能变 4.8 倍，value 没了。

所以我们网站后面应该做一个规则：

> 模型先出预测，但只在开跑前 10-5 分钟生成“最终建议”。

另外，第一场我建议少下或不下，因为当天场地偏差还没跑出来。跑完 1-2 场后，模型可以根据当天沙田/跑马地、草地/泥地、内外档、前置马是否有优势，调整后面场次。

### 助手（最终回复）

因为香港赛马主要是“彩池制”，不是你看到一个赔率就永远锁住。

简单说：

你现在看到的赔率，是当前大家已经投进去的钱算出来的。之后只要别人继续下注，赔率还会变。尤其最后 10 分钟，大资金进场最多，赔率变化最快。

所以太早下注有几个问题：

1. **赔率不稳定**
   早段可能显示 10 倍，看起来很香；但临开跑前大量资金买它，最后可能掉到 5 倍。你原本以为有 value，实际结算就没 value 了。

2. **信息还没完全确认**
   临场可能有退出马、场地变化、骑师/装备变化、赔率异动。太早下，等于少看了很多关键资料。

3. **市场还没成熟**
   太早的赔率池子钱少，容易失真。开跑前 10-5 分钟，投注量更大，赔率更接近真实市场判断。

但也不能最后 30 秒才下：

1. 系统可能卡、确认慢。
2. 赔率最后跳水，你来不及重新算。
3. 心态容易乱，变成追价。

所以最佳窗口是：**赛前 10 到 5 分钟**。
这个时间点信息基本齐，赔率也比较成熟，还留一点时间判断和执行。
