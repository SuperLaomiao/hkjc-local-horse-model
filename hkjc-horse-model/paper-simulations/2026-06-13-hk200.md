# HK$200 Paper Simulation - 2026-06-13 Sha Tin

This is a paper-only simulation. No real money is assumed, handled, or advised.

## Scope

- Meeting: 2026-06-13 Sha Tin twilight meeting, local Hong Kong races only
- Starting paper bankroll: HK$200
- Bet type: Win only
- Data source: official HKJC race card before the race and official HKJC result page after the race
- Freeze rule: model picks must be generated and saved before the first included race starts

## Staking Rules

- Default unit: HK$10 paper stake
- Maximum one paper bet per race
- Weekly spend cap: HK$200
- No chase betting after a loss
- No manual override after model output is frozen

## Recommendation Rules

The model may recommend a paper bet only when all selected gates pass:

- model win probability is at least 15%
- if market odds are available, expected return is positive:
  - `expected return = model probability * market win odds`
  - `edge = expected return - 1`
  - require `edge >= 0`
- if market odds are not available, use probability-only mode and mark the ticket as lower confidence
- if no horse clears the gates, the race is `PASS`

## Settlement Rules

After official HKJC results are published:

- If the recommended horse wins:
  - return = paper stake * official win odds
  - profit = return - paper stake
- If it does not win:
  - profit = -paper stake
- If the race is `PASS`:
  - stake = HK$0
  - profit = HK$0

## Weekly Report

Report after settlement:

- number of races
- number of paper bets
- wins / losses / passes
- total paper stake
- total paper return
- paper profit
- ROI
- model top-pick win rate
- notes on whether model probabilities were overconfident or underconfident

## Commands

Generate the dashboard with the HK$200 paper bankroll:

```bash
npm run hkjc:dashboard -- --input hkjc-horse-model/data/raw --bankroll 200 --minEdge 0 --minProbability 0.15 --maxStakePct 0.05
```

After official results are available, fetch and update:

```bash
npm run hkjc:fetch -- --date 2026-06-13 --course ST --races 1-11
npm run hkjc:dashboard -- --input hkjc-horse-model/data/raw --bankroll 200 --minEdge 0 --minProbability 0.15 --maxStakePct 0.05
npm test
```
